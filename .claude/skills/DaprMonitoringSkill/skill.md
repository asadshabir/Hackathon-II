# DaprMonitoringSkill

Monitors Dapr sidecars and Pub/Sub health.

## Capabilities
- Collect Dapr sidecar metrics
- Track state store latency
- Monitor pub/sub message delivery
- Publish observability events

## Integrations
- Dapr APIs
- Dapr State
- Pub/Sub channels

## Rules
- No direct infrastructure calls outside Dapr
- Stateless and idempotent
